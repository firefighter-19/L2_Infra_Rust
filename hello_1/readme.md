# Функционал `hello_1`

Приложение реализует консольный конвертер температур между шкалами Цельсия и Фаренгейта. Пользователь выбирает направление перевода и вводит значение температуры, после чего программа выводит результат.

## Основные возможности

- выбор конвертации из Цельсия в Фаренгейт или обратно;
- проверка корректности введённого варианта действия;
- повторное использование логики пересчёта через отдельные функции библиотеки.

## Как запустить

```bash
cargo run
```

## Тестирование
```bash
cargo test
```

Запустите команду из директории `hello_1`, затем следуйте подсказкам программы в терминале.
